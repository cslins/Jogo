# ECOAPP

Jogo de educação ambiental, Planejado para ser uma forma de ensinar crianças como fazer o descarte correto de resíduos recicláveis.

Esse projeto foi criado para avaliação em um curso de Laboratorio de Engenharia de Software, ofertado pela Faculdade de Computação, Universidade Federal do Pará, no período 2022-1.

# Como compilar o projeto
Caso esteja começamdo, siga as instruçoes disponiveis em:
 - instalar o Flutter: https://docs.flutter.dev/get-started/install
 - configurar um IDE: https://docs.flutter.dev/get-started/editor

Se você quiser usar um editor diferente, tudo bem, pule para a próxima etapa:
## **Android Studio and IntelliJ**
1. Localize a barra de ferramentas principal do Android Studio.
2. No seletor de destino, selecione um dispositivo Android para executar o aplicativo. Se nenhum estiver listado como disponível, selecione `Ferramentas` > `AVD Manager` e crie um lá.
3. Clique no ícone de execução na barra de ferramentas ou invoque o item de menu `Executar` > `Executar`.

Após a conclusão da compilação do aplicativo, você verá o aplicativo inicial no seu dispositivo.

## **Visual Studio Code**
1. Localize a barra de status do VS Code (a barra azul na parte inferior da janela):
2. Selecione um dispositivo na área Seletor de Dispositivos. Para obter detalhes, consulte [Alternar rapidamente entre dispositivos Flutter](https://dartcode.org/docs/quickly-switching-between-flutter-devices/).
    - Se nenhum dispositivo estiver disponível e você quiser usar um simulador de dispositivo, clique em `Nenhum dispositivo` e clique em Iniciar Simulador para iniciar um simulador.
     - Para configurar um dispositivo real, siga as instruções específicas do dispositivo na página [Install](https://docs.flutter.dev/get-started/install) do seu sistema operacional.
3. Invoque `Executa` > `Iniciar Depuração` ou pressione <kbd>F5</kbd>.
4. Aguarde a inicialização do aplicativo — o progresso é impresso na visualização do Console de depuração.

Após a conclusão da compilação do aplicativo, você verá o aplicativo inicial no seu dispositivo.

## **Terminal & editor**
1. Verifique se um dispositivo Android está em execução. Se nenhum for exibido, siga as instruções específicas do dispositivo na página [Install](https://docs.flutter.dev/get-started/install) do seu sistema operacional.
```
 flutter devices
```
2. Compile o aplicativo com o seguinte comando:
```
 flutter run
```

Após a conclusão da compilação do aplicativo, você verá o aplicativo inicial no seu dispositivo.
